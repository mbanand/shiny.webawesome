# Code of Conduct

This project is committed to providing a respectful, harassment-free
environment for everyone who participates in the repository, issue tracker,
documentation, and related project spaces.

## Expected behavior

Participants are expected to:

- communicate respectfully and professionally
- assume good faith while discussing technical disagreements
- focus criticism on ideas, code, and decisions rather than people
- be constructive in issues, pull requests, reviews, and discussions
- help keep the project welcoming to contributors with different backgrounds
  and levels of experience

## Unacceptable behavior

Unacceptable behavior includes, for example:

- harassment, threats, or intimidation
- derogatory or discriminatory language
- trolling, personal attacks, or sustained disruption
- sexualized language or imagery in project spaces
- publishing private information without permission
- other conduct that would reasonably make participation unsafe or hostile

## Maintainer responsibilities

Project maintainers may moderate discussions and contributions to keep the
project aligned with this Code of Conduct. This may include editing, removing,
or rejecting comments, issues, pull requests, commits, or other contributions
that violate these expectations.

## Reporting

If you experience or witness unacceptable behavior, please report it by
opening an issue or contacting the project maintainer directly:

- M. B. Anand <anand@owlcyberlabs.com>

Reports will be reviewed as promptly and fairly as possible.
